# Tron Legacy Theme for Zed

A dark theme for [Zed](https://zed.dev) inspired by the visual aesthetic of *Tron Legacy* — deep navy backgrounds, electric blue accents, and amber/orange syntax highlights.

## Colors

| Role | Color |
|---|---|
| Background | Deep navy `#14191f` |
| Electric blue (accents, types, keywords) | `#267fb5` |
| Amber/orange (functions, attributes) | `#ffb20d` |
| Red-orange (strings, labels) | `#ff410d` |
| Neon green (numbers, success) | `#c7f026` |
| Muted text | `#aec2e0` |

## Installation

### Via Zed Extension Gallery

Search for **Tron Legacy** in Zed's extension panel (`cmd+shift+x`).

### Manual

1. Clone this repo into your Zed extensions directory:
   ```
   git clone https://github.com/migs647/trong-legacy-theme-zed ~/.config/zed/extensions/tron-legacy-theme
   ```
2. Restart Zed and select **Tron Legacy** from the theme picker (`cmd+k cmd+t`).
